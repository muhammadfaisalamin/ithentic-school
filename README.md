# Ithentic School Management System

A high-performance School Management Dashboard built with the latest web
technologies.

## 🛠 Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Library:** React 19
- **Styling:** Tailwind CSS v4 (Oxide Engine)
- **Database:** PostgreSQL (Planned)
- **Testing:** Playwright (Planned)

## 🏗 Architecture

Using Next.js **Route Groups** `(dashboard)` to isolate the management interface
from public pages, ensuring a clean and modular folder structure.
